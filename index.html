DeepXY

DeepXY 是一个创新的 AI 对话系统，结合了 DeepSeek 的强大推理能力和其他模型的生成能力，提供了一个全新的 AI 对话体验。

🚀 主要特点

💡 双模型协作

结合 DeepSeek 的推理能力和 Qwen 的生成能力，实现更智能的对话。

🌊 流式输出

支持实时流式响应，提供更好的用户体验。

🔌 OpenAI 兼容

完全兼容 OpenAI API 格式，易于集成到现有系统。

🛠️ 快速开始

使用 Docker 快速部署：

docker-compose up -d

📚 支持的模型

  • DeepSeek R1
  • 通义千问 2.5
  • Google Gemini 2.0 Pro
  • Anthropic Claude 3 Sonnet
  • Meta Llama 2
  • Mistral Large
